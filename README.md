# Nebula Deploy (provisório)

Plataforma para deploy contínuo de aplicações Dockerizadas (MVP).

Estrutura inicial criada pelo assistente.

Como começar:

1. Backend:
   - cd backend
   - npm install
   - npm start

2. Frontend:
   - cd frontend
   - npm install
   - npm start

3. Worker:
   - cd worker
   - npm install
   - npm start
